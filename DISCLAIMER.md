# Disclaimer

## LLM-Generated Content

The manuscripts, code, and analysis results in this repository were produced
with substantial assistance from large language models (Claude, Anthropic).
This includes:

- **Case Study 1 (fully autonomous):** The entire pipeline output — methodology
  specification, implementation code, validation results, manuscript text,
  figures, and bibliography — was generated by Claude Code with zero human
  intervention. The only human input was the research brief describing the
  problem.

- **Case Study 2 (human-revised):** The initial manuscript was generated by
  Claude Code across multiple interactive sessions. A human researcher then
  revised the structure, verified data sources, corrected errors, and refined
  the exposition. Approximately 6 hours of human effort over ~3 weeks.

## Data Verification

The clinical trial dataset (`data/trials_data.csv`) was curated from published
sources and verified by cross-referencing against original publications. During
verification, 15 of 23 originally curated trials were found to contain errors
(wrong sources, incorrect rates, or fabricated MRD data). These were corrected,
and 2 trials were dropped entirely (MRD negativity never reported). The
corrected dataset contains 35 arms from 21 trials.

**Users should independently verify all data against primary sources before
using it for any clinical or regulatory purpose.**

## Statistical Results

The statistical methods and results presented in both case studies have not been
independently peer-reviewed. The self-grading scores (42/50 for both papers)
exhibit a known 1-4 point inflation compared to independent assessment (38-39/50).

## Reproducibility

The autonomous case study (KG-DAP) was produced in a single pipeline run on
2026-03-23. Due to the stochastic nature of LLM outputs, re-running the
pipeline from the same brief may produce a different method, different results,
and a different manuscript. The specific outputs in this repository represent
one realization of the pipeline.
